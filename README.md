# serial-oscilloscope
Oscilloscope Front-End and server for serial data

# run
node serial-test.js

# serve html
use python: python -m SimpleHTTPServer

# browser
open localhost:8000
Click connect button to start a WebSocket connection and start receiving 1 packet per second
